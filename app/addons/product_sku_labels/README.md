# Product SKU and Labels add-on

This add-on adds per-product controls for:
- enabling a custom SKU field
- displaying a custom SKU on the product detail page and product cards
- showing a product badge/label on catalog cards and product pages

## Installation
1. Copy the add-on folder to app/addons/product_sku_labels.
2. Install the add-on from the CS-Cart Admin panel.
3. The required database table will be created automatically during installation.

## What is included
- backend product fields for SKU and label settings
- frontend SKU display logic
- frontend product card label rendering
- import/export for the custom fields
